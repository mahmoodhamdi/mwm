## Getting Started

### Accessing the Admin Panel

1. Navigate to `https://your-domain.com/admin`
2. Enter your email and password
3. Click "Login"

### First-Time Setup

1. **Change Default Password**: Go to Profile > Change Password
2. **Configure Site Settings**: Go to Settings > General
3. **Set Up Contact Information**: Go to Settings > Contact
4. **Configure Social Links**: Go to Settings > Social

## Dashboard

The dashboard provides an overview of your website's activity:

### Metrics Cards

- **Total Visitors**: Monthly unique visitors
- **Page Views**: Total page views
- **Messages**: New contact messages count
- **Projects**: Total published projects

### Quick Actions

- View recent messages
- Quick access to content sections
- System notifications

## Services Management

### Adding a New Service

1. Go to **Services** in the sidebar
2. Click **Add New Service**
3. Fill in the form:
   - **Title (Arabic)**: العنوان بالعربية
   - **Title (English)**: English title
   - **Description (Arabic)**: الوصف بالعربية
   - **Description (English)**: English description
   - **Icon**: Select from icon library
   - **Image**: Upload service image
   - **Category**: Select category
   - **Featured**: Toggle to show on homepage
   - **Active**: Toggle to publish
4. Click **Save**

### Managing Categories

1. Go to **Services > Categories**
2. To add: Click **Add Category**
3. To edit: Click the edit icon next to a category
4. To delete: Click the delete icon (only if no services assigned)

### Reordering Services

1. Go to **Services**
2. Click and drag the handle (≡) to reorder
3. Changes save automatically

## Projects/Portfolio

### Adding a New Project

1. Go to **Projects** in the sidebar
2. Click **Add New Project**
3. Fill in the form:
   - **Title**: Project name (Arabic/English)
   - **Description**: Detailed description
   - **Client**: Client name
   - **Completion Date**: When the project was completed
   - **Category**: Select category
   - **Technologies**: Add technologies used
   - **Images**: Upload project images (first image is featured)
4. Click **Save as Draft** or **Publish**

### Project Status

- **Draft**: Not visible on the public site
- **Published**: Visible on the portfolio page
- **Featured**: Highlighted on the homepage

### Image Guidelines

- Recommended size: 1200x800 pixels
- Maximum file size: 5MB
- Supported formats: JPG, PNG, WebP
- First uploaded image becomes the cover image

## Team Management

### Adding Team Members

1. Go to **Team** in the sidebar
2. Click **Add Team Member**
3. Fill in the form:
   - **Name**: Full name (Arabic/English)
   - **Position**: Job title (Arabic/English)
   - **Bio**: Short biography
   - **Photo**: Upload profile photo
   - **Social Links**: LinkedIn, Twitter, GitHub
   - **Order**: Display order
4. Click **Save**

### Photo Guidelines

- Recommended size: 400x400 pixels (square)
- Professional headshot preferred
- Supported formats: JPG, PNG

## Contact Messages

### Viewing Messages

1. Go to **Messages** in the sidebar
2. Messages are sorted by date (newest first)
3. Click on a message to view details

### Message Status

- **New**: Unread message (blue badge)
- **Read**: Message has been viewed
- **Replied**: Response sent to sender
- **Archived**: Moved to archive
- **Spam**: Marked as spam

### Actions

- **Star**: Mark important messages
- **Reply**: Send email response
- **Archive**: Move to archive
- **Mark as Spam**: Report spam

### Bulk Actions

1. Select multiple messages using checkboxes
2. Choose action from dropdown:
   - Mark as Read
   - Mark as Unread
   - Star
   - Unstar
   - Archive
   - Mark as Spam
   - Delete
3. Click **Apply**

## Newsletter

### Viewing Subscribers

1. Go to **Newsletter** in the sidebar
2. View all email subscribers
3. Filter by status (Active, Unsubscribed)

### Exporting Subscribers

1. Click **Export**
2. Choose format (CSV or Excel)
3. Download file

### Managing Subscriptions

- **Unsubscribe**: Remove subscriber from list
- **Reactivate**: Restore unsubscribed user

## Content Management

### Editing Page Content

1. Go to **Content** in the sidebar
2. Select the page to edit:
   - Home Page
   - About Page
   - Services Page
   - Contact Page
3. Edit sections:
   - Hero section
   - Features section
   - CTA sections
4. Click **Save**

### Managing Translations

1. Go to **Translations**
2. Search for specific text
3. Edit Arabic or English versions
4. Click **Save**

### Adding FAQs

1. Go to **Content > FAQs**
2. Click **Add FAQ**
3. Enter question and answer in both languages
4. Set display order
5. Click **Save**

## User Management

### Adding New Users

1. Go to **Users** (Admin only)
2. Click **Add User**
3. Fill in details:
   - Name
   - Email
   - Password
   - Role
4. Click **Create User**

### User Roles

- **Admin**: Full access to all features
- **Editor**: Can manage content but not users or settings
- **Viewer**: Read-only access to dashboard

### Managing Permissions

1. Go to **Users > Roles**
2. Click on a role to edit
3. Toggle permissions:
   - Services: Read, Create, Update, Delete
   - Projects: Read, Create, Update, Delete
   - Team: Read, Create, Update, Delete
   - Messages: Read, Update, Delete
   - Settings: Read, Update
4. Click **Save**

## Best Practices

### Content Guidelines

1. **Keep it bilingual**: Always add content in both Arabic and English
2. **Use high-quality images**: Optimize before uploading
3. **SEO-friendly titles**: Use descriptive titles
4. **Regular updates**: Keep content fresh and current

### Image Optimization

1. Resize images before uploading
2. Use WebP format when possible
3. Add alt text for accessibility
4. Keep file sizes under 500KB when possible

### Security

1. Use strong passwords (min 12 characters)
2. Enable two-factor authentication
3. Log out when not in use
4. Don't share credentials

## Troubleshooting

### Common Issues

#### Can't Upload Images

- Check file size (max 5MB)
- Verify file format (JPG, PNG, WebP)
- Clear browser cache

#### Changes Not Showing

- Wait for cache to clear (up to 1 minute)
- Hard refresh (Ctrl + F5)
- Check if content is published/active

#### Login Issues

- Verify email and password
- Check if account is active
- Contact admin for password reset

## Appendix

### Supported Image Formats

| Format | Max Size | Use Case                      |
| ------ | -------- | ----------------------------- |
| JPEG   | 5MB      | Photos, complex images        |
| PNG    | 5MB      | Logos, icons, transparency    |
| WebP   | 5MB      | Modern browsers, smaller size |
| SVG    | 500KB    | Icons, logos (vector)         |

### API Rate Limits

- 100 requests per 15 minutes per user
- File uploads: 10 per minute
- Bulk actions: 50 items per request
